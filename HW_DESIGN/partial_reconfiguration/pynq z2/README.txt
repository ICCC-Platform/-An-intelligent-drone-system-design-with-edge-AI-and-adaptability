# This is DEMO

In order to install it to your PYNQZ-Z2, connect to the board, open a terminal and type: numeric key 0~6  

option:  
0 - Exit  
1 - Configure XTEA  
2 - Configure RTEA  
3 - Configure AES  
4 - Configure DES  
5 - Configure Blank  
6 - Run encrypt and decode  

Notice: Make sure that you configure 1~5 before you select option 6 for the test
